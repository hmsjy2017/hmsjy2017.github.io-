开源音乐神器

发表于 2020-05-13
  • 做最好的自己
首页 音乐

GitHub 上有 a great number of 很棒的音乐工具，堪称神器。这里就介绍以下我使用过的 3 款。

# 1.MKOnlineMusicPlayer

GitHub：https://github.com/mengkunsoft/MKOnlineMusicPlayer


MKOnlineMusicPlayer 是作者孟坤开发的在线音乐播放器，支持多平台音乐搜索、播放、下载🌝，有实时同步的歌词，还可以同步自己的网易云音乐歌单
而且界面也很美观，适配移动端，仅占数百 KB 空间。搭建简便，只需 PHP+Nginx 即可食用。注意，使用 PHP 7.2 搭建可能会出现歌单无法加载的问题。
欢迎访问我搭建好的地址：http://music.iamsjy.com:88/player/

# 2. 音乐搜索器

GitHub：https://github.com/maicong/music


音乐搜索器是作者麦葱开发的支持多平台音乐搜索、播放、下载的工具。与前者孟坤在线播放器不同的是，音乐搜索器的功能侧重音乐搜索及下载，支持更多的平台：网易云、QQ 音乐、虾米音乐、百度音乐、喜马拉雅 FM、5Sing 等，播放功能则不够完善。占空间极小，仅几十 KB。同样搭建简便，只需 PHP+Nginx。
欢迎访问我搭建好的地址：http://music.iamsjy.com:88/

# 3.Unlock Music

GitHub：https://github.com/ix64/unlock-music

Unlock Music 是在一款在浏览器中解锁加密的音乐文件的工具。通常情况，付费音乐都是加密的文件格式，下载以后只能在会员期内在指定播放器内播放，并且会员到期之后如不续费将无法播放。Unlock Music 可以将加密的音乐转化成 mp3 格式，支持批量解锁、拖放解锁、在线播放、PWA、多线程，可谓十分人性化。Unlock Music 是在浏览器中解锁，无需上传到服务器。由于 PWA 等特性，支持离线使用。可以搭建在 GitHub Pages、Gitee Pages 等，亦提供了离线使用的版本。
欢迎使用：https://iamsjy.gitee.io/music/
