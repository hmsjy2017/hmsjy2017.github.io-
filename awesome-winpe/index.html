优秀、纯净、可靠的 WinPE

以下 PE 由个人或团队开发，无广告、无收费、无劫持，可以说是用爱发电。
每一款都很优秀，值得一试（当然优秀的还不止这么多）。~~ 坚决抵制老毛桃、大白菜、U 深度、电脑店等不可靠的商业 PE。~~ 网络功能除了微 PE 应该都支持。按发现的时间先后排序。

# 微 PE

http://www.wepe.com.cn/download.html
微 PE 团队开发。纯净无捆绑（用过的都知道）
缺点：不支持网络

# 一个不知道叫什么好的 U 盘启动工具集

https://359303267.github.io/PE/
某初中生坚持制作这个 U 盘启动工具集到大学毕业，如今由于个人原因已停止更新。
优点：

  • 支持网络
  • 集成许多常用工具
  • 系统加载到内存中运行（也就是 RAMOS），使用很流畅
    缺点：
  • 占用空间较大


# FirPE

https://firpe.cn/page-247
某 00 后创立。
它具有简约、易操作等特点，使用起来十分人性化。以 U 盘作为使用载体，空间更为充分，携带更为方便。同时整合各种装机必备工具，有效提高系统安装效率。

# Edgeless

官网：https://home.edgeless.top
部分开源：https://github.com/Cnotech/Edgeless-Project
背景：
Edgeless 是 Cno 在高二就开始酝酿的一个 PE 项目，高三下半学期（对就是高考前）一直在策划 Edgeless 的相关内容，如今已上大学。

Edgeless 的创立旨在试图替代 WTG 的部分应用场景 。
特点：

  • 强大的插件功能（能兼容其他 PE）
  • 优雅的主题功能
  • 基于 Win10 的 PE 核心

# Hikari PE

文档：https://hikaripe-sc.hikaricalyx.com
和微 PE 这样的行业标杆不同，Hikari PE 在保证功能稳定的情况下，紧跟时代前沿，探索使用 PE 维护 PC 的更多可能性。

# MSDN，我告诉你

https://msdn.itellyou.cn
新版网站：https://next.itellyou.cn
镜像全部来自微软 MSDN，没有任何修改。
缺点：

# UUP dump

https://www.uupdump.net
轻松地从 Windows 更新服务器下载 UUP 文件。
优点：

  • 自由选择版本
  • 多线程，下载速度更快
    缺点：制作时长很大程度受到机器性能影响

# UUP Generation Project

https://uup.rg-adguard.net
优点：同 UUP dump
缺点：

  • 制作时长很大程度受到机器性能影响
  • 该网站国内访问缓慢

# Latest10

https://latest10.win
服务器定时检测新 Build，只要有更新则触发服务器进行转换 / 生成镜像，并上传到云服务。镜像无修改。
优点：

  • 下载速度快
  • 镜像无需制作，直接下载
